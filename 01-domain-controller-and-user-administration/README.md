# Domain Controller Deployment & User Administration (Windows Server 2022)

## Overview
This project documents the deployment of a Windows Server 2022 Domain Controller and the creation of user accounts, groups, and administrative roles within Active Directory. The goal is to practice real-world identity management tasks performed by system administrators.

This project includes:
- Installing and configuring Windows Server 2022
- Deploying Active Directory Domain Services (AD DS)
- Promoting the server to a Domain Controller
- Creating Organizational Units (OUs)
- Creating domain users and security groups
- Assigning administrative roles and permissions
- Validation tests to confirm configuration works

---

## Objectives
- Install Windows Server 2022 and configure static networking  
- Create a new forest and domain (`stevenlab.local`)  
- Create OUs for Users, Computers, and Admins  
- Create standard user accounts  
- Create privileged accounts and delegate permissions  
- Validate login, policy application, and administrative access
